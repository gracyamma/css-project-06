## 🎨 Customization Guide

### 🖌️ Change Fonts

Edit Tailwind config:

```js
fontFamily: {
  poppins: ["Poppins", "sans-serif"],
  orbitron: ["Orbitron", "sans-serif"],
}
```

### 🌈 Modify Colors

Replace Tailwind classes:

```html
bg-black → bg-blue-500
text-slate-400 → text-gray-500
```

### 🖼️ Replace Images

Update files inside `/assets` folder.
